# Marvel

A good night application for track and share sleeping time.

## Stacks

- Ruby v3.2.2
- Ruby on Rails v7.0.8.7
- Postgres
- Redis
- Sidekiq

## How to start

Before starting the app please run command below

```
bundle install

rails db:create

rails db:migrate

rails db:seed
```

## How to test
